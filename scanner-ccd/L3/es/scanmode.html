Modo de escaneo#

Modo clave#

En este modo, el módulo comienza a leer el código después de presionar el botón disparador y deja de leerlo después de leerlo exitosamente o de soltar el botón disparador.

Modo continuo#

La luz roja del módulo permanece encendida durante mucho tiempo. Cuando pasa un código de barras, el módulo lo lee automáticamente. El mismo código de barras no se puede leer repetidamente a menos que se extraiga y se lea nuevamente.

Modo de retardo de tecla#

Una vez realizada la configuración, la luz roja del módulo permanecerá encendida durante 3 segundos y se apagará si no se lee el código de barras en 3 segundos, o se apagará después de que se lea correctamente el código de barras.
