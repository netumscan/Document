Verwenden Sie das MQTT-Protokoll zum Übertragen von Daten#

Erstellen Sie einen Proxy-Server (Broker).#

  • Verwenden Sie einen Server, um MQTT Broker zu erstellen, und die vom Scanner hochgeladenen Barcodes werden über diesen Server übertragen.

Serververbindungsinformationen

Bemerkung

  • Der Scanner stellt standardmäßig über Port 1883 eine Verbindung zum MQTT-Broker her.

  • Der vom Scanner abonnierte Themeninhalt ist die Geräte-ID.
