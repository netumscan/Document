Utilisation du client TCP pour transmettre des données#

Configuration rapide#

   

Note

Lorsque vous utilisez un logiciel tiers pour créer un client TCP, vous pouvez utiliser cette option pour la configuration.

Logiciel de démonstration#

  • Cliquez sur le lien ci-dessous pour télécharger le logiciel de démonstration : Netum Wi-Fi TCP Server

Établir une connexion#

  1. Téléchargez le logiciel de démonstration et décompressez-le dans un dossier.

  2. Ouvrez le logiciel de démonstration, saisissez les informations WiFi et les informations du serveur TCP auxquelles le scanner doit se connecter, puis utilisez un scanner pour scanner le code QR ci-dessous.

  1. Cliquez pour démarrer le service ou démarrez le serveur TCP via la programmation.

  2. Attendez que le scanner se connecte activement au service TCP Server du logiciel.

Note

  • L’adresse de l’hôte est l’adresse IP de l’ordinateur sur lequel se trouve le service TCP Server.

  • Le port hôte est le numéro de port ouvert par le service TCP Server.
