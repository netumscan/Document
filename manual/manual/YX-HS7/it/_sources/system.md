### USB COM

```{figure} ../../../media/881001133..png
:align: center
:width: 300px

USB COM
```

```{note}
连接电脑，扫描[USB Keyboard](#usb-keyboard)条码，即可把设备配置为 USB 键盘模式。  
在应用端软件需求串口的使用环境下，扫描[USB COM](#usb-com)，识别为 USB COM 类型，需要用户安装驱动，（此条码默认配置为：115200 波特率，8位数据位，无校验位，1 位停止位）。
```
