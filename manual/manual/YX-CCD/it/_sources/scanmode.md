# 扫描模式

## 按键模式
在该模式下，模块在按下触发按键后开始读码，读码成功或松开触发按键后停止读码。


```{figure} ../../../media/013300.png
:align: center
:width: 300px

按键模式 `默认`
```

## 连续模式
模块红光灯处于长亮状态，有条码经过时，模块自动读取条码。相同条码不可重复读取除非移开重新读取。

```{figure} ../../../media/013304.png
:align: center
:width: 300px

连续模式
```

## 按键延时模式
设置成功后，模块红光灯长亮3S，超时3S未读码灯灭，或者读取条码成功后灯灭。

```{figure} ../../../media/013301.png
:align: center
:width: 300px

按键延时模式
```
