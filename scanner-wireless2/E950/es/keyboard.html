Configuración del caso#

Idioma del teclado#

  • Si está utilizando un teclado francés, escanee el código de barras del comando «Francés».

  • Si usa un teclado estadounidense, puede ignorar este paso (el teclado estadounidense se usa de manera predeterminada en China).

  • Si los datos escaneados están ilegibles, es posible que el idioma del método de entrada del teclado no corresponda con el idioma del escáner.

Teclado universal internacional ALT[2]#

[1]

Configuración del idioma TH, el módulo de escaneo también debe configurarse en salida TH.

[2]

El teclado universal internacional ALT solo se aplica a los sistemas Windows y no se ve afectado por la configuración del teclado de la computadora, pero sí afectará la velocidad de transmisión.
