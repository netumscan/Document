Mode de fonctionnement#

Si la zone de travail est en dehors de la portée du récepteur 2,4G ou du signal Bluetooth, vous pouvez activer le mode de stockage du scanner en suivant les étapes décrites ci-dessous. Dans ce mode, toutes les données numérisées seront stockées directement dans le scanner. Avant de télécharger les données, celles-ci seront enregistrées de manière permanente dans le scanner afin que vous puissiez télécharger les données lorsque le scanner se trouve dans la portée du signal de réception.

Mode normal#

Note

Passez en mode normal et scannez le code pour télécharger les données instantanément.

Mode de stockage#

Note

Définir le mode de stockage. La numérisation du code ne téléchargera pas les données immédiatement, mais les données seront stockées dans le scanner.

Téléchargement de données[1]#

Note

Téléchargez les données stockées dans le scanner.

Mode de sauvegarde automatique activé[2]#

[1]

Pendant le téléchargement des données, l’analyse à nouveau de la commande « Téléchargement des données » annulera/arrêtera le téléchargement des données.

[2]

En mode de transmission sans fil, lorsque le mode de stockage automatique est activé, le code-barres sera automatiquement stocké lorsque la transmission échoue, et les données stockées peuvent être obtenues via « Téléchargement de données ».
