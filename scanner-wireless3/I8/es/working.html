Modo de trabajo#

Si el área de trabajo está fuera del alcance del receptor 2.4G o de la señal Bluetooth, puede activar el modo de almacenamiento del escáner siguiendo los pasos que se describen a continuación. En este modo, todos los datos escaneados se almacenarán directamente en el escáner. Antes de cargar los datos, estos se guardarán de forma permanente en el escáner para que pueda cargarlos cuando el escáner esté dentro del rango de recepción de la señal.

Modo normal#

Nota

Establezca el modo normal y escanee el código para cargar datos instantáneamente.

Modo de almacenamiento#

Nota

Establecer el modo de almacenamiento. Al escanear el código no se cargarán los datos inmediatamente, pero estos se almacenarán en el escáner.

Carga de datos[1]#

Nota

Sube los datos almacenados en el escáner.

Almacenamiento total#

Nota

Comprueba cuántos registros están almacenados.

Borrar el código de barras almacenado#

Nota

Borre los datos almacenados en el escáner.

Modo de guardado automático desactivado#

Modo de guardado automático activado[2]#

[1]

Durante la carga de datos, al escanear nuevamente el comando «Carga de datos» se cancelará/detendrá la carga de datos.

[2]

En el modo de transmisión inalámbrica, cuando el modo de almacenamiento automático está activado, el código de barras se almacenará automáticamente cuando falle la transmisión y los datos almacenados se pueden obtener a través de «Carga de datos».
